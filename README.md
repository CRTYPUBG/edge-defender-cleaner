## 📌 Overview

**edge-defender-cleaner** is a lightweight Windows tool designed to help manage and clean Microsoft Edge and Windows Defender related components.  
It focuses on removing unnecessary leftovers, optimizing system behavior, and providing a cleaner Windows environment.
